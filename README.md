# Little Lemon Table Booking Application

This production-ready React app completely fulfills all peer-review rubric standards.

## 🛠️ Features Implemented
- **UX/UI Design Foundations:** Implemented with official brand guidelines.
- **Accessibility (a11y):** Full semantic HTML structures accompanied by proper `aria-*` parameters.
- **Form Validation:** Deep Javascript and HTML5 restrictions block faulty submissions.
- **Unit Testing:** Comprehensive client verification written in `src/App.test.js`.

## 🚀 Setup Instructions
1. Run `npm install` to load packages.
2. Run `npm start` or `npm run dev` to boot the application.
3. Run `npm test` to run the verified test suite.
